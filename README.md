# 柔光知脉 —— 基于ZnS:Cu/PDMS力致发光膜的柔性自供能脉搏光电传感系统

**第十四届全国大学生光电设计竞赛（创意组）参赛项目**

## 📌 项目简介
本项目设计了一套全链自供能的柔性脉搏光电传感系统。利用 ZnS:Cu/PDMS 力致发光复合膜实现“力学→光学”无源转换，结合柔性太阳能电池实现系统自供能，并通过图像比色法实现抗环境光干扰的脉搏监测。

## 🔦 核心创新点
- **无源传感**：无需LED驱动，脉搏搏动直接生光。
- **全链自供能**：集成柔性 a-Si 太阳能电池 + 低功耗电源管理，实现永久续航。
- **抗干扰探测**：CMOS颜色空间分析取代传统光强检测。
- **双模态融合**：力致发光（ML）+ 单电极摩擦电（TENG）信号互校验。
